# 〆トラ ブランドガイド

## カラーシステム

### ブランドカラー（2色）

〆トラのブランドは **インディゴ（`--brand`）** と **ダークネイビー（`--ink`）** の 2 色で構成されます。

| 役割 | 変数 | 値 | 用途 |
|---|---|---|---|
| **アクションカラー** | `--brand` | `#4f46e5` (Indigo 600) | ボタン・リンク・フォーカスリング・ハイライト |
| **アイデンティティカラー** | `--ink` | `#0f0d1a` | ロゴ・シール・LP の暗色セクション背景 |
| 暗背景用アクセント | `--brand-light` | `#a5b4fc` (Indigo 300) | 暗い背景上のテキストアクセント |
| アクション hover | `--brand-600` | `#4338ca` | ボタン hover 状態 |
| 薄いアクション面 | `--brand-50` | `#eef2ff` | 通知カード・入力フォーカス背景 |

### サーフェスカラー

| 変数 | 値 | 用途 |
|---|---|---|
| `--paper` | `#f7f5f0` | ページ背景（温かみのある白） |
| `--paper-2` | `#ede9df` | セクション背景・セカンダリ面 |
| `--card` | `#ffffff` | カード・モーダル背景 |

### テキスト階層（インク）

| 変数 | 値 | 用途 |
|---|---|---|
| `--ink` | `#0f0d1a` | 見出し・主要テキスト |
| `--ink-2` | `#2c2740` | 本文テキスト |
| `--ink-3` | `#6b6580` | サブテキスト・ラベル |
| `--ink-4` | `#9a93ad` | プレースホルダー・補足情報 |

## ファンクショナルカラー

ブランドカラーとは区別する。**データの状態表示専用**に使い、装飾や情報強調には用いない。

### 緊急度（urgency）

締切までの残り時間を示す 4 段階。ダッシュボードの緊急度インジケーター・LP の緊急度デモのみで使用。

| 緊急度 | 変数 | 値 | 条件 |
|---|---|---|---|
| 期限切れ | `--u-overdue` / `--u-overdue-bg` | `#d23a3a` / `#fdecec` | 締切超過 |
| 当日 | `--u-today` / `--u-today-bg` | `#e9651c` / `#fff1e8` | 当日中 |
| 3日以内 | `--u-soon` / `--u-soon-bg` | `#b0871f` / `#fff7da` | 72 時間以内 |
| 通常 | `--brand` / `--brand-50` | — | 4日以上 |

### ステータス

| ステータス | 変数 | 値 | 状態 |
|---|---|---|---|
| 完了 | `--s-done` / `--s-done-bg` | `#1f8a5b` / `#e6f5ed` | submitted / done |

### 選考種別タグ

選考の種別を素早く識別するためのパステルカラー。装飾・ブランディングには使わない。

| 種別 | 背景色 | テキスト色 | Tailwind クラス |
|---|---|---|---|
| ES | `#e9defb` | `#5b3a93` | `.kind-es` |
| 説明会 | `#d8eef0` | `#1f6168` | `.kind-briefing` |
| 面接 | `#fde0c8` | `#a04316` | `.kind-interview` |
| その他 | `--paper-2` | `--ink-2` | `.kind-other` |

## 使用ルール

### ✅ 正しい使い方

```
CTA ボタン（LP）       : bg=--ink  → hover: bg=--brand（スライドアップ）
CTA ボタン（アプリ）   : bg=--brand → hover: bg=--brand-600
フォーカスリング       : shadow=--brand-50 + border=--brand
強調テキスト（明背景） : color=--brand
強調テキスト（暗背景） : color=--brand-light
通知カード背景         : bg=--brand-50
```

### ❌ 避けるべき使い方

```
緊急度色を装飾に使う   : 見出しや区切りに --u-today（オレンジ）を使わない
種別色を背景に使う     : カードやセクションの背景に種別パステルを使わない
インクカラーをボタンに : アプリ内の通常ボタンは --ink でなく --brand を使う
直書きのカラー値       : #e9651c 等のハードコードより CSS 変数を優先
```

## LP と アプリのカラー使い分け

| コンテキスト | 主色 | 補色 |
|---|---|---|
| LP（editorial） | `--ink` (dark sections) | `--brand` (accent/hover) |
| アプリ（functional） | `--brand` (actions) | `--ink` (identity/logo) |

LP は編集的・エディトリアルなデザインのため、`--ink` を支配色として使う（ヒーロー、マーキー、CTA セクションの背景）。アプリはクリーンな操作画面のため、`--brand` を主要なアクションカラーとして使う。
