## dashboardページ

下記を参照してdashboardページのテキストあるいは機能を更新してください。

### 現在の表示テキスト

- 現在の表示テキスト: @docs/text/dashboard.md

### 更新内容

- H1（状態によって変化）を以下のように変更
  | 状態 | テキスト |
  |---|---|
  | アイテムなし | `締切登録はありません` |
  | 期限切れあり | `現在の締切登録：{N}件（{M}件期限切れ。）` |
  | 通常 | `現在の締切登録：{N}件` |

- KPI タイル（アイテムが1件以上ある場合に表示）を以下のように変更
  | タイル | ラベル（mono） | 値 |
  |---|---|---|
  | 登録中 | `登録中` | 数値 |
  | 期限切れ | `期限切れ` | 数値（期限切れがある場合は赤色） |
  | 完了 | `完了` | 数値（緑色） |

- HeroCard（次の未対応アイテムがある場合に表示）のグリーティング行を削除

- HeroCard（次の未対応アイテムがある場合に表示）のカウントダウンとアイテム情報は以下の形式で表示（期限超過時は赤色）
  またこのカードの時間表示はリアルタイム表示したい。

```
次の締め切りまで
{企業名} {種別タグ}
{時} h {分} m {秒} s
（プログレスバー：残り14日を100%として表示）
```
