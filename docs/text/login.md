## 左カラム（PC のみ表示）

### ブランドロゴ
```
〆  トラ
```

### キャッチコピー
```
（H1）  締切を、
        出し忘れない
        人生に。

（サブコピー）  ES・説明会・面接の締切を一元管理し、
               締切前にメールで通知します。
```

### チェックリスト（3項目）
- 締切72時間前・24時間前・3時間前に通知
- クレジットカード不要で10件まで無料
- メールアドレスだけで30秒登録

## 右カラム（フォーム）

### ページタイトル・説明
```
（H1）  締切を、出し忘れない人生に。

（サブ）  メールアドレスを入力するとログインリンクを送ります。
```

### LoginForm

#### 通常状態
| 要素 | テキスト |
|---|---|
| 入力欄プレースホルダー | `you@example.com` |
| 送信ボタン | `マジックリンクを送信` |
| 送信中ボタン | `送信中…` |

#### 送信完了状態
```
メールを送信しました

{メールアドレス} 宛にログインリンクを送りました。
リンクの有効期限は 30 分です。届かない場合は迷惑メールをご確認ください。

（リンク）  別のアドレスで送り直す
```

#### エラーメッセージ（URLパラメータ）
| エラーコード | 表示テキスト |
|---|---|
| `expired` | リンクが期限切れです。もう一度メールを送信してください。 |
| `invalid` | 無効なリンクです。再度ログインしてください。 |
| `server` | サーバーエラーが発生しました。しばらく後にお試しください。 |

#### APIエラーメッセージ（フォールバック）
- `送信に失敗しました`
- `ネットワークエラーが発生しました`
