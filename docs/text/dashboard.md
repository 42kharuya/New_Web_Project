# ダッシュボード `/dashboard`

> ファイル: `src/app/dashboard/page.tsx`, `src/app/dashboard/DeadlineList.tsx`, `src/app/dashboard/HeroCard.tsx`

---

## ページヘッダー

### グリーティング（時間帯で変化）

| 時間帯 | テキスト |
|---|---|
| 5〜11時 | `GOOD MORNING` |
| 12〜16時 | `GOOD AFTERNOON` |
| 17〜21時 | `GOOD EVENING` |
| 22〜4時 | `GOOD NIGHT` |

右端には `{今日の日付}（例: 5月14日(水)）` を表示。

### H1（状態によって変化）
| 状態 | テキスト |
|---|---|
| アイテムなし | `締切アイテムはありません` |
| 期限切れあり | `今日の締切は{N}件期限切れ。` |
| 通常 | `今日の締切は{N}件あります。` |

---

## KPI タイル（アイテムが1件以上ある場合に表示）

| タイル | ラベル（mono） | 値 |
|---|---|---|
| 未対応 | `TODO` | 数値 |
| 期限切れ | `OVERDUE` | 数値（期限切れがある場合は赤色） |
| 完了 | `DONE` | 数値（緑色） |

---

## HeroCard（次の未対応アイテムがある場合に表示）

### グリーティング行
```
{GOOD MORNING etc.} · {今日の日付}
```

### カウントダウン
```
{値} {単位}
```
- 単位: `日` / `時間` / `期限切れ`（期限超過時は赤色）

### アイテム情報
```
{企業名}
{種別タグ}
（プログレスバー：残り14日を100%として表示）
```

## Free プラン上限バナー（10件に達した場合のみ表示）

Free プランの上限（10件）に達しています。　Pro へのアップグレードで無制限に追加できます。

（ボタン）  Pro にアップグレード

## 締切一覧（DeadlineList）

### セクションヘッド
```
締切一覧  {件数バッジ}
```

### フィルターチップ
| キー | ラベル |
|---|---|
| `all` | 全て |
| `todo` | 未対応 |
| `done` | 済 |

### アイテムカード内テキスト

| 要素 | テキスト |
|---|---|
| 企業名 | `{companyName}`（完了・辞退時は取り消し線） |
| 種別タグ | `ES` / `説明会` / `面接` / `その他` |
| 日時 | フォーマット済み日時文字列 |
| メモ | `{memo}`（設定時のみ表示、1行） |
| ステータスピル（未対応） | `未対応` |
| ステータスピル（提出済） | `提出済` |
| ステータスピル（完了） | `完了` |
| ステータスピル（辞退/中止） | `辞退/中止` |
| 編集ボタン | aria-label: `{企業名}を編集` |
| 削除ボタン | aria-label: `{企業名}を削除` |

### 削除確認

| 要素 | テキスト |
|---|---|
| 実行ボタン | `削除` |
| キャンセルボタン | `×` |

### エラー表示
```
⚠ {エラーメッセージ}
```

エラーメッセージの例：
- `ステータスの更新に失敗しました`
- `削除に失敗しました`

### 空状態（アイテムが0件）
```
〆
締切アイテムがまだありません

まずは 2件登録 して、締切管理をスタートしましょう

（ボタン）  + 締切を登録する
```

### フィルター空状態
```
該当する締切はありません
```
