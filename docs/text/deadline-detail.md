## HeroCard（ダッシュボードと共通）

グリーティング・カウントダウン・企業名・種別タグ・プログレスバーを表示。
→ 詳細は [`dashboard.md`](./dashboard.md) の HeroCard セクションを参照。

## ステータス 4 択ボタン

| ステータス | ラベル |
|---|---|
| todo | `未対応` |
| submitted | `提出済` |
| done | `完了` |
| canceled | `辞退/中止` |

エラー時:
```
{エラーメッセージ}
```
例: `更新に失敗しました` / `ステータスの更新に失敗しました`

## 詳細行（dl リスト）

| ラベル | 値 |
|---|---|
| 締切日時 | フォーマット済み日時 |
| 種別 | `ES` / `説明会` / `面接` / `その他` |
| リンク | `公募リンク ↗`（設定時のみ） |
| メモ | {memo テキスト}（設定時のみ） |
