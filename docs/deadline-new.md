## DeadlineForm（mode: create）

### Free 上限エラーバナー（10件に達した場合のみ）

```
Free プランの登録上限（10件）に達しました
Pro にアップグレードすると締切アイテムが無制限になります。

（ボタン）  Pro にアップグレード →
```

### グローバルエラー（フォールバック）
```
保存に失敗しました。もう一度お試しください。
```
その他エラー例：
- `締切アイテムが見つかりません。`（編集時404）
- `この操作は許可されていません。`
- `ネットワークエラーが発生しました。`

## フォームフィールド

### 企業名
| 要素 | テキスト |
|---|---|
| ラベル | `企業名 *` |
| プレースホルダー | `例: 株式会社テクノロジー` |
| バリデーションエラー | `企業名は必須です` |

### 種別（タイル選択）
| 要素 | テキスト |
|---|---|
| ラベル | `種別 *` |
| バリデーションエラー | `種別は必須です` |

#### 種別タイル 4枚
| アイコン | ラベル | 説明文 |
|---|---|---|
| 📝 | ES | エントリーシート |
| 📅 | 説明会 | イベント・会社説明会 |
| 🗣️ | 面接 | 面接・選考 |
| 📌 | その他 | その他の選考 |

### 締切日時
| 要素 | テキスト |
|---|---|
| ラベル | `締切日時 *` |
| バリデーションエラー | `締切日時は必須です` |

#### クイック時間チップ
| チップ | 設定値 |
|---|---|
| `今夜` | 当日 23:59 |
| `明日` | 翌日 23:59 |
| `3日後` | 3日後 23:59 |
| `1週間後` | 7日後 23:59 |

### リンク
| 要素 | テキスト |
|---|---|
| ラベル | `リンク（任意）` |
| プレースホルダー | `https://example.com/job` |

### メモ
| 要素 | テキスト |
|---|---|
| ラベル | `メモ（任意）` |
| プレースホルダー | `選考メモ・注意事項など` |
